# britishcouncil-eda

This is exploratory data analysis of British Council programme I,as part of the team of 3, led in Andhra Pradesh. This reports intends to answer following questions -
1. Explanation about what the project is.
2. Pre-assessment tests called OPT which were taken to understand the levels of a sample of students. I, along with British Council Experts, tuned the online learning part accordingly.
3. Understanding the grades received in pre-assessment tests.
4. After 1 year of online & offiline training, around 21000 students underwent final test called APTIS. This report aims to analyse their results and.
5. Understand the performance of colleges where students took both OPT and APTIS tests. This will help us understand the improvement in learning outcomes at the college level. This is only for colleges where students gave both OPT (pre-assessment) and APTIS (final-assessment)tests.
